# Hary Pictures

A modern portfolio website for Hary Pictures, built with TypeScript and modern web technologies.

## Overview

This repository contains the source code for the Hary Pictures portfolio website. It showcases photography work and provides a professional online presence for the Hary Pictures brand.

## Tech Stack

- **TypeScript** (96.2%) - Primary language for type-safe development
- **JavaScript** (2.6%) - Supporting scripts and legacy code
- **Other** (1.2%) - Build tools and configuration

## Features

- Responsive design for mobile, tablet, and desktop
- Fast performance optimization
- Professional portfolio showcase
- User-friendly navigation

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Ghostfaceki11err/Hary-Pictures.git
cd Hary-Pictures
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open your browser and navigate to `http://localhost:3000` (or the port specified in your configuration)

### Build for Production

```bash
npm run build
# or
yarn build
```
